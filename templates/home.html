FAQ (CLICK)
- What version is server/world running on?

> Server is always at the most recent version of game which is currently 1.19.2. World was started at 1.17.1 and some chunks around 0/0 may belong to this version. You can join from any version from 1.18.2 up to 1.19.2.

- Is the server cracked?/I can't join/Can't verify

> The server is only for official MC accounts. If you can't join make sure that your device is connected to network, ip isn't wrong and you have official MC account. Also depends on what it says on disconnect screen.

- Is there plugins/tpa/claims/dynmap etc?

> Server is 100% vanilla and there are no plugins affecting gameplay.

- Is hacking allowed?

> Hacking isn't forbidden but discouraged. Temporary ban can be implemented in case of total abuse.

- Is griefing allowed?

> You can do whatever you want as far as others not preventing you from doing it obviously.

- Is KeepInventory enabled?

> No. And never will be. The Server is vanilla. KeepInventory is ruining many aspects of vanilla gameplay such as PvP and survival in general.

- I'm falling into the void, what do I do?

> That means that you have bad connection to server try reconnecting. You may have bad Wi-Fi or server might be overloaded.

- What is world seed?

> Seed can be exploited in many ways and that's why it's not and won't be public. Seed is useless to have since generation seeds are different on server.

- Is the server whitelisted?

> Nope, it's open for everyone.

- Is the server bedrock or java?

> It's java.

- Does server reset?

> Nope. The server will never reset. Map is permanent. Only unused chunks are being cleared once in a while.

- Is the nether roof opened?

> No it's not. It's disabled due to abuse and being imbalanced. Farms and travelling are still possible without it. Don't go there, it deals void damage.
